...над полем битвы кружили стаи воронов. Многие доблестные воины полягли в этом сражении, многие еще продолжали сражаться.
"Если так и дальше будет продолжаться, мы просто перебьем друг друга. И в этой битве не будет победителей - проиграют все." - так размышлял сэр Рональд, наблюдая неутешительную картину перед собой.
"Я должен принять важное решение. Я знаю цену такого поступка, но сейчас только это может спасти нас всех..."
Когда-то давно, когда он был весьма склонен к авантюрам и поиску приключений, он отправился охотиться на ведьму, за голову которой была обещана огромная награда. Хитрая бестия смогла спасти свою жизнь, уговорив рыцаря принять от неё дар - пузырек с кровью вампира. Эта кровь, влитая в уста умирающего человека, могла вернуть его к жизни в облике вампира. Неужели день, когда ему придется использовать это, настал?..
Похоже, только так можно выиграть эту битву.
Сэр Рональд начал склоняться над едва живыми телами своих рыцарей, которые лежали возле него. Каждому из них он говорил:
- "Пей. Тебя ждет новая жизнь..."

Итак, у нас есть 3 типа солдат: Warrior, Knight и Defender. Давайте сделаем сражения армий еще более эпичными и добавим новый тип юнитов - Vampire!
Vampire должен быть наследником класса Warrior и иметь дополнительный параметр - vampirism, который помогает ему исцелять себя. Когда вампир наносит удар другому юниту, то восставнавливает себе здоровье в размере +50% от нанесенного урона (защита соперника уменьшает наносимый вампиром урон и, соответственно, получаемое лечение).
Базовые параметры класса Vampire:
health = 40
attack = 4
vampirism = 50%

Пример: 

chuck = Warrior()
bruce = Warrior()
carl = Knight()
dave = Warrior()
mark = Warrior()
bob = Defender()
mike = Knight()
rog = Warrior()
lancelot = Defender()
eric = Vampire()
adam = Vampire()
richard = Defender()
ogre = Warrior()

fight(chuck, bruce) == True
fight(dave, carl) == False
chuck.is_alive == True
bruce.is_alive == False
carl.is_alive == True
dave.is_alive == False
fight(carl, mark) == False
carl.is_alive == False
fight(bob, mike) == False
fight(lancelot, rog) == True
fight(eric, richard) == False
fight(ogre, adam) == True

my_army = Army()
my_army.add_units(Defender, 2)
my_army.add_units(Vampire, 2)
my_army.add_units(Warrior, 1)
    
enemy_army = Army()
enemy_army.add_units(Warrior, 2)
enemy_army.add_units(Defender, 2)
enemy_army.add_units(Vampire, 3)

army_3 = Army()

army_3.add_units(Defender, 4)

army_4 = Army()
army_4.add_units(Vampire, 3)
army_4.add_units(Warrior, 2)

battle = Battle()

battle.fight(my_army, enemy_army) == False
battle.fight(army_3, army_4) == True

Входные данные: воины и армии.

Выходные данные: результат сражения (True или False).

Как это используется: Для разработки компьютерных игр.

Предусловие: 4 типа солдат
