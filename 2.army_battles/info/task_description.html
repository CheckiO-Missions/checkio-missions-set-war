In the previous mission - Warriors you've learned how to make a duels between 2 warriors. Good work! But let's move to the more epic thing - the armies! In this mission your task is to add a new class and a function to the existed. The new class is Army which will has the method add_units() - for the adding chosen amount of units to the army. Also you need to create a battle() function, which will choose the strongest army.
The battles will go by the next principles:
at the first will be the duel between the first unit of the first army and the first unit of the second army. As soon as one of the duelists will be dead, the second unit from the army, which has lost the warrior, will continue the duel, and the survived warrior is also continue but with his current (not full) health. The battle goes on while every army has at least 1 live unit. After the battle the function should return the result: True if the first army wins or False in the other case.

Example: 

chuck = Warrior()
bruce = Warrior()
carl = Knight()
dave = Warrior()

fight(chuck, bruce) == True
fight(dave, carl) == False
chuck.is_alive == True
bruce.is_alive == False
carl.is_alive == True
dave.is_alive == False

my_army = Army()
my_army.add_units(Knight, 3)
    
enemy_army = Army()
enemy_army.add_units(Warrior, 3)

army_3 = Army()
army_3.add_units(Warrior, 20)
army_3.add_units(Knight, 5)
    
army_4 = Army()
army_4.add_units(Warrior, 30)

battle(my_army, enemy_army) == True
battle(army_3, army_4) == False

Input: warriors and armies.

Output: result of the battle (True or False).

How it is used: For the computer games development.

Precondition: 2 types of units
