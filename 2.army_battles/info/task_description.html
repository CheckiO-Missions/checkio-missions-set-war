In the previous mission - Warriors you've learned how to make a duels between 2 warriors. Good work! But let's move to the more epic thing - the armies! In this mission your task is to add a new class and a function to the existed. The new class is Army which will has the method add_unit() - for the adding chosen unit to the army. Also you need to create a battle() function, which will choose the strongest army.
The battles will go by the next principles:
at the first will be the duel between the first unit of the first army and the first unit of the second army. As soon as one of the duelists will be dead, the second unit from the army, which has lost the warrior, will continue the duel, and the survived warrior is also continue but with his current (not full) health. The battle goes on while every army has at least 1 live unit. After the battle the function should return the result: "First army wins!" or "Second army wins!".
If there is a situation when all units in both armies have died, the function should return "All died...".

Example: 

bob = Warrior(20, 4)
carl = Warrior(30, 5)
jimbo = Warrior(50, 1)
han = Warrior(40, 3)
luc = Warrior(60, 2)

my_army = Army()
my_army.add_unit(bob)
my_army.add_unit(carl)
my_army.add_unit(jimbo)

enemy_army = Army()
enemy_army.add_unit(han)
enemy_army.add_unit(luc)

neutral_minions = Army()
neutral_minions.add_unit(jimbo)
neutral_minions.add_unit(carl)

battle(my_army, enemy_army) == "First army wins!"
battle(neutral_minions, enemy_army) == "Second army wins!"

Input: warriors and armies.

Output: the winner.

How it is used: For the computer games development.

Precondition: health > 0
attack > 0
